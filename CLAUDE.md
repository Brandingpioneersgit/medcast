# MedCasts

## Architecture
- **Stack**: Next.js 15 (App Router) + PostgreSQL 16 + Drizzle ORM + Tailwind CSS + next-intl
- **Entry**: `src/app/[locale]/layout.tsx` (locale-aware root with RTL support)
- **DB Schema**: `src/lib/db/schema.ts` — 30+ tables (hospitals, doctors, treatments, specialties, conditions, packages, leads, referrals, translations, FAQs, blog)
- **Queries**: `src/lib/db/queries.ts` — all DB access functions
- **i18n**: 8 locales (en, ar, ru, fr, pt, bn, tr, hi), RTL for Arabic, content translations via DB `translations` table
- **API**: `/api/v1/inquiry` (lead capture), `/api/v1/search` (global search)
- **Deploy target**: 2 VPS (US + India) + Cloudflare CDN
